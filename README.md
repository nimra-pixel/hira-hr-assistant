# 💼 HIRA — HR Intelligence & Recruitment Assistant

> AI-powered dual-mode HR platform for job seekers and HR managers — powered by Groq + Llama 3 + Streamlit

## Features

### 🧑‍💼 Job Seeker Mode
- 💬 AI career coach chat with memory
- 📄 Resume analyzer — ATS score, skill detection, improvements
- 🎤 Mock interview simulator with AI feedback
- 📊 Career intelligence report + salary range + learning roadmap

### 🏢 HR Manager Mode
- 💬 HR operations assistant chat
- 📝 Job description writer
- ❓ Interview question bank generator
- 📊 Candidate evaluation rubric
- 📄 Offer letter templates
- 📋 Onboarding checklists
- ⚖️ Pakistan labor law reference

## Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy
```bash
git init && git add . && git commit -m "HIRA HR Assistant"
git remote add origin https://github.com/nimra-pixel/hira-hr-assistant.git
git push -u origin main
```
Streamlit secrets: `GROQ_API_KEY = "gsk_..."`
